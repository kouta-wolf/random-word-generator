# ランダム名言ジェネレーター

## 概要
ジェネレーターボタンを押すとプログラマ向けの有名な名言をカードに入れてくれるWebアプリです。

## リンク
[random-quote-generator](https://kouta-wolf.github.io/random-word-generator/)

## 学習の目的
- Reactの `useState` と `useEffect` の基本的な使い方と挙動の理解
- Tailwind CSS を用いた効率的なスタイリング
- 外部JSONデータ（疑似API）のフェッチと非同期処理の実装

## 最終出力
- GitHub pagesへ公開

## 使用技術
- React (Vite)
- Tailwind CSS
- GitHub Pages
